# Kelly Coins 🪙

一个专为4岁小朋友Kelly设计的家务奖励游戏！完成任务赚取金币，用金币换取心爱的奖励。

## 启动服务器

```bash
cd kelly-coins
npm install
npm start
```

服务器会在 http://localhost:3000 启动。

## 访问地址

| 页面 | 本机地址 |
|------|---------|
| Kelly的页面（孩子用）| http://localhost:3000 |
| 家长管理面板 | http://localhost:3000/parent |

## iPad / 手机访问（局域网）

1. 在电脑上打开命令提示符，输入 `ipconfig`
2. 找到 **IPv4 地址**，例如 `192.168.1.100`
3. 在 iPad 或手机浏览器输入：
   - Kelly的页面：`http://192.168.1.100:3000`
   - 家长面板：`http://192.168.1.100:3000/parent`

> 确保电脑和 iPad 连接在**同一个 WiFi 网络**下。

## 功能说明

### Kelly的页面（孩子）
- **任务** 📋 — 点击完成的任务，立即获得金币，有金币雨动画和音效
- **奖励** 🎁 — 浏览可兑换的奖励，金币够了就能点"换！"
- **存钱罐** 🐷 — 查看余额、距离下个目标的进度、最近记录

### 家长面板
- **总览** — 余额、今日获得、待审批数量
- **待审批** — 批准或拒绝Kelly的兑换请求（拒绝会退还金币）
- **任务管理** — 修改任务金币数量、启用/停用任务
- **奖励管理** — 修改奖励价格、启用/停用奖励
- **历史记录** — 最近50条收支记录
- **手动调整** — 奖励或扣除金币（附备注）

## 数据存储

数据保存在 `kelly-coins.db`（SQLite文件），重启服务器不会丢失数据。
